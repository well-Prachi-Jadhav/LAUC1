Virtual Mathematics Laboratory – Exact Differential Equations

Aim

To study the concept of exact and non-exact differential equations and to develop an interactive virtual laboratory that determines the exactness of a given first-order differential equation of the form M(x,y)dx + N(x,y)dy = 0. The experiment also aims to help students understand the role of integrating factors and the systematic procedure for obtaining the general solution.

Theory

A first-order differential equation of the form M(x, y) dx + N(x, y) dy = 0 is called an exact differential equation if there exists a scalar function F(x, y) such that:

dF = (∂F/∂x) dx + (∂F/∂y) dy = M dx + N dy

This implies that:

M = ∂F/∂x    and    N = ∂F/∂y

A necessary and sufficient condition for exactness is that the mixed partial derivatives are equal, provided the functions are continuous in the given region:

∂M/∂y = ∂N/∂x

If the given differential equation is exact, the solution can be obtained by integrating M(x, y) with respect to x, treating y as a constant. The resulting function is then differentiated partially with respect to y and compared with N(x, y) to determine the remaining function of y.

The final solution of an exact differential equation is expressed implicitly as:

F(x, y) = C

Many practical differential equations are not exact in their original form. In such cases, an integrating factor (μ) is introduced to convert the equation into an exact one.

If the expression (∂M/∂y − ∂N/∂x) / N is a function of x alone, then an integrating factor μ(x) exists. Similarly, if (∂N/∂x − ∂M/∂y) / M is a function of y alone, then an integrating factor μ(y) exists.

Multiplying the original differential equation by the appropriate integrating factor converts it into an exact equation, which can then be solved using the standard procedure.

Exact differential equations are widely used in engineering and applied sciences such as thermodynamics, fluid mechanics, and electrical engineering.

Procedure

  1. Input the functions M(x,y) and N(x,y).
  2. Compute ∂M/∂y and ∂N/∂x.
  3. If ∂M/∂y = ∂N/∂x, the equation is exact.
  4. Integrate M with respect to x to obtain F(x,y).
  5. Add the missing function of y by comparing with N.
  6. If non-exact, test for integrating factor μ(x) using (∂M/∂y − ∂N/∂x)/N.
  7. Similarly test μ(y) using (∂N/∂x − ∂M/∂y)/M.
  8. Multiply by μ and solve as an exact equation.
  9. Display the solution F(x,y) = C.

Solved Example

Problem:
Check whether the differential equation
(2xy + y2) dx + (x2 + 2xy) dy = 0
is exact. If exact, find its general solution.

Step 1: Identify M(x, y) and N(x, y)

M(x, y) = 2xy + y2
N(x, y) = x2 + 2xy

Step 2: Compute the partial derivatives

∂M/∂y = ∂/∂y (2xy + y2) = 2x + 2y
∂N/∂x = ∂/∂x (x2 + 2xy) = 2x + 2y

Step 3: Check the exactness condition

Since ∂M/∂y = ∂N/∂x, the given differential equation is exact.

Step 4: Integrate M with respect to x

∫(2xy + y2) dx = x2y + xy2 + g(y)

Step 5: Differentiate with respect to y and compare with N

∂/∂y (x2y + xy2 + g(y)) = x2 + 2xy + g'(y)

Comparing with N(x, y) = x2 + 2xy, we get:
g'(y) = 0 ⇒ g(y) = C

Step 6: Write the general solution

x2y + xy2 = C

Final Answer:
The given differential equation is exact and its general solution is x2y + xy2 = C.
